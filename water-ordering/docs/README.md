**内容**：

### [04-业务逻辑说明](./04-业务逻辑说明.md)

**内容**：

- 数据流转图

### [05-数据库文档](./05-数据库文档.md)
**用途**：了解数据库表结构和维护方法  
**适合**：后端开发、数据库管理员  
**内容**：
- 所有表结构说明

- 索引信息
- 数据库初始化方法
- 维护和备份指南

**相关文件**：
- `backend/database/init.sql` - 数据库初始化SQL（手动执行）
- `backend/database/test_data.sql` - 测试数据SQL（可选）
- `backend/database/fix_phone_verified.sql` - 字段修复SQL（如需要）
